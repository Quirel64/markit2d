================================================================================
                        PIXEL GRID STUDIO - ROADMAP
================================================================================

A browser-based pixel art editor built with React 19 + TypeScript + Vite.
Designed for both PC and mobile with a unified layout.

================================================================================
                         FILE STRUCTURE
================================================================================

src/
  App.tsx              - Main component (~2000 lines). Contains all UI layout,
                         tool logic, canvas rendering, menu state, project
                         management, and event handlers.

  App.css              - All styles (~960 lines). Flex layout, floating panels,
                         tool sidebar, canvas stage, responsive rules.

  types/index.ts       - Type definitions: Tool, ShapeType, Pixel, MenuId,
                         ProjectMeta, Viewport, Selection, FloatingSelection,
                         PathOptions, HslColor.

  constants/index.ts   - Constants: CANVAS_SIZE (128), VIEW_SIZE (768),
                         GRID_PRESETS, BRUSH_PRESETS, EXPORT_SCALES,
                         TOOLS (8 tools), MENUS (3 tabs), SHAPE_PRESETS,
                         PALETTE (28 colors), storage keys.

  utils/
    canvas.ts          - makeBlankPixels, clonePixels, indexOf, clamp.
                         Pixel grid data operations.
    viewport.ts        - clampViewport, getDistance, getCenter.
                         Zoom/pan math and pointer helpers.
    color.ts           - hexToRgb, hexToHsl, hslToHex, shiftColor.
                         Color conversion and manipulation.
    pathfinding.ts     - A* pathfinding for the path tool.
    project.ts         - encodeProject, decodeProject, listProjects,
                         loadProject, saveProject, deleteProject,
                         createProject, updateProject, isCanvasBlank.
                         Multi-project localStorage management.


================================================================================
                      COMPLETED FEATURES (v0.1)
================================================================================

--- LAYOUT (Current) ---

  [x] Top-docked split layout
      - Menu tabs (Tools/Grid/Color) inline in the topbar
      - Left sidebar with tool icons (PC only, >=768px)
      - Canvas fills remaining space via flex
      - Quick pins bar at the bottom
      - Settings panel slides from the right

  [x] Floating menu panels
      - Click a menu tab -> panel appears ON TOP of canvas
      - Slide-down + fade animation (200ms ease)
      - Click backdrop to dismiss
      - Max-height 50svh, scrolls internally

  [x] Responsive design
      - PC: sidebar + floating panels + full topbar
      - Mobile: no sidebar, compact topbar with horizontal scroll
      - Settings menu adapts to screen width

--- TOOLS ---

  [x] View tool (pan/zoom)
  [x] Pencil tool
  [x] Eraser tool
  [x] Fill tool (flood fill)
  [x] Eyedropper tool
  [x] Select tool (rectangle select, copy/cut/paste, flip, rotate)
  [x] Path tool (A* pathfinding, diagonal toggle, color-crossing toggle)
  [x] Shape tool (rectangle, circle, line with live preview)

  All tools accessible from:
    - Left sidebar (PC)
    - Tools menu in floating panel
    - Quick pins bar (pinned tools)

--- CANVAS ---

  [x] 128x128 pixel grid (CANVAS_SIZE)
  [x] Display at 768x768 (VIEW_SIZE)
  [x] CSS container-query based sizing (never clips)
  [x] Grid lines toggle
  [x] Grid size presets (8x8 to 64x64)

--- ZOOM & PAN ---

  [x] Always-on pinch-to-zoom (mobile, any tool)
  [x] Always-on scroll-to-zoom (PC, any tool)
  [x] View tool for pan (single finger/mouse drag)
  [x] Zoom range: 1x to 16x

--- COLOR ---

  [x] 28-color preset palette
  [x] Shade ring (8 variants + center)
  [x] Custom color picker (native input)
  [x] Pin/unpin colors
  [x] Pinned colors palette
  [x] Eyedropper tool to pick from canvas

--- PROJECT MANAGEMENT ---

  [x] Multi-project storage (localStorage)
